Utilities Form

* Click Utilities button and you will see five options, the Add FAQ, Add Request Type, Back Up, Restore, and Help.

* To Add FAQ, click the Add FAQ button it will redirect you to a page that will allow you to select a certain category to create questions and their respective answers. Click the add button to create the question and answer.

* To Add Request Type, click the Add Request Type button it will redirect you to a page that will allow you to create request types. You will need to enter or type a request type then click the add button to create the request type.

* To restore, click the Restore button and it will redirect you to a form that you need to choose a file location then click the restore in order to restore your file.

* To see some guide, click the Help button where you can see some intructions about how to use the certain function of the system.
